# ⚡ Sigilcraft

> Reality is programmable. Consciousness is the code.

The living sigil library of the Sparkverse. A subdomain of [The First Spark](https://thefirstspark.shop).

**Live at:** https://sigilcraft.thefirstspark.shop

## What's inside

- `/` — Landing page
- `/library.html` — Full sigil library, organized by Selector Model layer (Physics / Metaphysical / Relational / Temporal)
- `/generator.html` — The Forge: generates a unique, deterministic sigil from any name + intention using canvas + TFS visual DNA
- `/daily.html` — One sigil per day, seeded by the calendar date so every visitor sees the same sigil
- `/sigils/` — Individual sigil pages with meaning, activation ritual, downloads, and Spark counter

## Stack

Pure HTML / CSS / vanilla JS. No build step, no framework. GitHub Pages friendly.

## Deploy

This repo uses a `CNAME` file to resolve `sigilcraft.thefirstspark.shop`. Add a CNAME DNS record at the domain registrar pointing `sigilcraft` → `thefirstspark.github.io`.
